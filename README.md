# Project Description

This project is a network-based currency exchange office system developed in C#.

The system consists of:

* WCF Web Service
* WPF Client Application
* SQLite Database
* Integration with the NBP API

The application retrieves current exchange rates from the National Bank of Poland (NBP) API and allows users to perform currency exchange operations.

Implemented functionality:

* User creation
* Currency conversion
* Exchange rate retrieval
* Transaction storage
* SQLite database integration

## How to Run

1. Open the solution in Visual Studio.
2. Restore NuGet packages if required.
3. Start the CurrencyExchangeService project.
4. Start the CurrencyExchangeWpfClient project.
5. Create a user.
6. Perform currency exchange operations.

## Project Structure

* CurrencyExchangeService – WCF service and business logic
* CurrencyExchangeWpfClient – client application
* CurrencyClient – console client
* ExchangeOffice.db – SQLite database
